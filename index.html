Política de Privacidade - CardioCare

1. Dados que Coletamos

Coletamos os seguintes tipos de dados para garantir o funcionamento da aplicação:

a) Dados Pessoais

  • Nome completo
  • Data de nascimento
  • Gênero
  • Dados de contato (e-mail, telefone)

b) Dados de Saúde

  • Altura e peso
  • Pressão arterial (sistólica e diastólica)
  • Frequência cardíaca
  • Glicemia e colesterol
  • Hábitos de vida (fumo, consumo de álcool, atividade física)

2. Como Utilizamos os Dados

Os dados coletados são utilizados para prever e monitorar o risco de doenças cardiovasculares, enviar lembretes personalizados e facilitar consultas médicas.

3. Compartilhamento de Dados

Garantimos que os dados coletados não serão vendidos, alugados ou compartilhados com terceiros, exceto nos casos descritos nesta política.
